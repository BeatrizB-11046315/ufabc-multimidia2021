Descrição dos procedimentos:

Linha 1: Leitura da imagem colorida. image_path é o "caminho" até o arquivo, que está no Google Drive. 'avatares.jpg', é o nome do arquivo. Imagem colorida definida por cv.IMREAD_COLOR

Linhas 2-3: redimensionamento da imagem, pelo cv.resize.

Como parâmetros: (Imagem a ser redimensionada, variável em que deve ser armazenado o resultado, fator de escala para eixo horizontal, fator de escala para eixo vertical, interpolação pelo cv.INTER_CUBIC)

Analogamente para Linhas 4-6, porém com a imagem dos integrantes do grupo.

Linhas 7-10: Exibição das imagens, com o OpenCV

imgavatares = cv.imread(os.path.join(image_path,'avatares.jpg'),cv.IMREAD_COLOR)

imgavatares30 = cv.resize(imgavatares, None, fx=0.30, fy=0.30, interpolation = cv.INTER_CUBIC)

imgavatares20 = cv.resize(imgavatares, None, fx=0.20, fy=0.20, interpolation = cv.INTER_CUBIC)

imggrupo = cv.imread(os.path.join(image_path,'foto-grupo.png'),cv.IMREAD_COLOR)

imggrupo40 = cv.resize(imggrupo, None, fx=0.40, fy=0.40, interpolation = cv.INTER_CUBIC)

imggrupo10 = cv.resize(imggrupo, None, fx=0.10, fy=0.10, interpolation = cv.INTER_CUBIC)

cv2_imshow(imgavatares30)

cv2_imshow(imgavatares20)

cv2_imshow(imggrupo40)
